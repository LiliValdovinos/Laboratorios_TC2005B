Ejercicios de JavaScript

Explicación del problema elegido

El problema elegido es un convertidor de grados Celsius a Fahrenheit. Se implementa en un objeto en JavaScript con un método que recibe una temperatura en Celsius y devuelve la conversión a Fahrenheit.

Respuestas a las preguntas teóricas

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  • Ambos usan una sintaxis inspirada en C, lo que significa que comparten estructuras como if, for, while, etc.
  • Ambos soportan programación orientada a objetos, aunque de manera diferente (Java usa clases y herencia, mientras que JavaScript utiliza prototipos).
  • Java se utiliza en servidores (backend), mientras que JavaScript es el principal lenguaje de desarrollo web en navegadores (frontend).

2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

  • getFullYear(): Devuelve el año actual.
  • getMonth(): Devuelve el mes (0-11).
  • getDate(): Devuelve el día del mes.
  • getDay(): Devuelve el día de la semana (0 para domingo).
  • getHours(): Devuelve la hora actual, desde 0 a 23 (24 horas).

3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5)

  • push(): Agrega un elemento al final del arreglo.
  • pop(): Elimina el último elemento del arreglo.
  • shift(): Elimina el primer elemento del arreglo.
  • unshift(): Agrega un elemento al inicio del arreglo.
  • forEach(): Ejecuta una función para cada elemento del arreglo.

4. ¿Cómo se declara una variable con alcance local dentro de una función?

Se usa let o const dentro de una función. Ejemplo:

function ejemplo() {
    let x = 5; // Variable local, solo existe dentro de esta función
}

5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • Pueden ser modificadas accidentalmente por cualquier función, causando errores. (Sobreescritura)
  • Ocupamos más memoria, ya que la variable no se elimina al terminar la función.
  • El código es más difícil de depurar porque muchas funciones pueden afectar la misma variable.
