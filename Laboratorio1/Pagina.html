Preguntas A Responder

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la red global de computadoras(red que conecta redes de ordenadores de todo el mundo),
mientras que la WWW es un sistema de información basado en hipervínculos (servicio que presta Internet.).

¿Cuáles son las partes de un URL?

  • Protocolo
  • Sudominio
  • Dominio
  • TLD
  • Subcarpeta
  • Slug
  • Parámetros

¿Cuál es el propósito de los métodos HTTP?

  • GET: Solicita datos del servidor.
  • HEAD: Obtiene los encabezados de la respuesta sin el cuerpo.
  • POST: Envía datos al servidor para su procesamiento.
  • PUT: Actualiza un recurso existente en el servidor.
  • PATCH: Modifica parcialmente un recurso en el servidor.
  • DELETE: Elimina un recurso en el servidor.

¿Qué método HTTP se usa al enviar formularios?

El método POST, porque envía los datos de manera segura al servidor.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

El método GET, ya que permite solicitar recursos sin modificar información en el servidor.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

El código 200 indica que la solicitud fue exitosa y que el servidor devolvió la información sin errores.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Sí, porque el error 404 indica que un recurso no fue encontrado, lo que puede deberse a un enlace roto o un archivo eliminado.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Sí, ya que un error 500 indica un problema interno en el servidor, generalmente debido a fallos en el código.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)?

Significa que ya no se recomienda su uso porque puede ser eliminado en futuras versiones del estándar HTML.

Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

  • <font>
  • <center>
  • <applet>
  • <acronym>

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

  • HTML5 introduce nuevas etiquetas semánticas.
  • Mejor soporte para multimedia con etiquetas <audio> y <video>.
  • Elimina elementos obsoletos como <font>.
  • Soporta API para almacenamiento local y aplicaciones web.

¿Qué componentes de estructura y estilo tiene una tabla?

  • Contenedor de tabla (<table>)
  • Encabezados (<thead>)
  • Cuerpo (<tbody>)
  • Columnas (<th>)
  • Filas (<tr>)
  • Pie de tabla (<tfoot>)

¿Cuáles son los principales controles de una forma HTML5?

  • Input
  • Textareaa
  • Button
  • Form
  • Label
  • Fieldset
